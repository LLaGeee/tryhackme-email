Training Modules in the Pathway

Discover the comprehensive modules created by in-market experts with over 25 years of combined AWS experience:

Introduction to Amazon Web Services

Learn the unique characteristics associated with cloud technologies and core fundamentals.

Introduction to Identity and Access Management (IAM)

Set the foundation for attacking and defending by understanding how permissions work on AWS.

Attacking and Defending Core Services

Explore security misconfiguration on commonly used AWS services including EC2, S3, VPC, KMS and more.

Attacking and Defending Serverless

Work with Serverless components within AWS (including Lambda and Step functions) to understand how they can be exploited.

Identity and Access Management (IAM) Privilege Escalation

Put knowledge to practice by covering key concepts required to enumerate, exploit and persist across IAM.

Suitable for all Cloud Practitioners
